Almasilos, S.A.

El sistema desarrollado para Almasilos, S.A. está diseñado a la medida para gestionar y optimizar el control de las operaciones de la empresa, utilizando una arquitectura basada en microservicios desplegados en un clúster de Kubernetes. Este enfoque permite escalabilidad, resiliencia y una fácil administración de los servicios. Se implementaron contenedores Docker tanto para la aplicación web como para los servicios back-end, mientras que la base de datos MySQL asegura un manejo eficiente de la información. Además, el sistema incorpora un balanceador de carga para distribuir el tráfico de manera equitativa entre los nodos, garantizando un rendimiento óptimo, incluso durante picos de demanda. Por último, se ha configurado un firewall avanzado para proteger el sistema contra accesos no autorizados y asegurar un control efectivo del tráfico.
